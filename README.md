# Homebrew Tap for Vision|Pipe

## Installation

```bash
brew tap visionpipe/visionpipe
brew install --cask visionpipe
```

## About

Vision|Pipe gives your LLM eyes. Capture your screen, annotate with voice, text, or drawing, and paste full visual context into any AI in one keystroke.

Learn more at [github.com/VisionPipe/visionpipe](https://github.com/VisionPipe/visionpipe).
